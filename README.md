## Run Locally

Clone the project

```bash
  git clone https://github.com/emmanuelronoh/E-commerce-website
```

Go to the project directory

```bash
  cd E-Commerce-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
